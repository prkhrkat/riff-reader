# Riff Reader

An iOS app that listens to music, analyzes the guitar melody, and displays notes in a Guitar Hero-style game interface for learning and practice.

## Features

- **Real-time Audio Analysis**: Listen to music through the microphone and extract guitar melodies
- **Pitch Detection**: On-device pitch detection using AVFoundation and Accelerate framework
- **Note Visualization**: Display notes on a 6-string guitar fretboard with animated gameplay
- **Playback Verification**: Listen to user playing and provide feedback
- **Cloud Enhancement**: Optional cloud-based analysis for complex musical passages

## Requirements

- iOS 17.0+
- Xcode 15.0+
- Swift 5.9+
- Microphone permission
